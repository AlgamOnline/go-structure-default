## 🚀 Getting Started

### 1️⃣ Initialize Module
```bash
go mod init github.com/AlgamOnline/go-structure-default
```

### 2️⃣ Install Dependencies
```bash
go mod tidy
```

### 3️⃣ Run Application
```bash
go run cmd/app/main.go
```
